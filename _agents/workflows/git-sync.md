---
description: Sincronización automática con GitHub (stage, commit y push).
---

// turbo-all
1. Agregar todos los cambios locales:
```powershell
git add .
```

2. Realizar el commit con un mensaje descriptivo:
```powershell
git commit -m "Sincronización automática desde workflow"
```

3. Subir a la rama principal:
```powershell
git push origin main
```
